# war-card-game-app
![](./images/war_card-game-js.png)

## Premise
Play the card game "War" with the computer.

First one who loses all of their cards loses the game.

* Player = Yellow cards
* Computer = Red cards

## Description
This is a basic Javascript app with CSS styling.
